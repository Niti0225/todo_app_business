# Planivo Website Upload

Dieser Ordner enthält die statische Webseite für Planivo.

## Hochladen

Alle Dateien und Ordner aus diesem Verzeichnis hochladen:

- `index.html`
- `privacy.html`
- `terms.html`
- `faq.html`
- `imprint.html`
- `site.css`
- `site.js`
- `assets/app-icon.png`

## Vor dem Launch ersetzen

- Vollständige Impressumsdaten: Anschrift, ggf. Unternehmensform, Registerangaben, Umsatzsteuer-ID.
- App-Store-Links auf der Startseite.
- Optional echte App-Screenshots ergänzen, falls du später zusätzliche Store- oder Galerie-Bilder zeigen möchtest.
- Finale Preise und Abo-Produkte aus App Store, Google Play und RevenueCat.
- Rechtliche Prüfung von Datenschutz, Nutzungsbedingungen und Impressum.
- Store Privacy Labels und Google Play Data Safety Angaben mit dem finalen Build abgleichen.
